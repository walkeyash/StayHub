A full-stack Airbnb-style property listing and discovery platform, built from scratch using Node.js, Express, and MongoDB, following the MVC (Model-View-Controller) architecture.
🔗 Live demo: https://stayshub.onrender.com

Features
🔐 User Authentication — secure signup/login using Passport.js with session-based auth
🏘️ Listing Management (CRUD) — create, view, edit, and delete property listings
🖼️ Image Uploads — listing images stored and served via Cloudinary
🔍 Search & Filter — search listings by keyword and filter by category (Trending, Rooms, Mountains, Camping, etc.)
🗺️ Interactive Maps — each listing's location is geocoded and displayed on a Mapbox map
⭐ Reviews & Ratings — logged-in users can leave star ratings and comments on listings
💬 Flash Messages — real-time success/error feedback on user actions
📱 Responsive UI — built with EJS templating and Bootstrap for a clean, mobile-friendly layout
